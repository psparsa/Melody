### System requirment:
- RAM: < 1gb
- Node.js: <= 16.20

### Starting the dev sever:

```bash
npm install
npm run dev
```

### Inspecting UI components:
```bash
npm run storybook
```

### Running linter:

```bash
npm run lint
```

### Formatting the project codes:
```bash
# to check which files need formatting
npm run prettier:check
npm run prettier
```


### Deployment with docker:
```bash
docker-compose up
```
